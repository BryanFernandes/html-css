Desafio das imagens

O objetivo desse desafio é colocar um ícone de favoritos e uma imagem no conteúdo da página, como está no PDF do exercício que pode ser acessado clicando aqui

Para inserir uma imagem basta abrir um parágrafo e usar a tag img. Depois disso, coloque o endereço em src e uma descrição da imagem no alt.
